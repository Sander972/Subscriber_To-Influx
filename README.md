# Subscriber_To-Influx

Console app that subscribe to a topic and send the mqtt msg to an InfluxDb

## Getting Started

Run inside the project folder:
```
npm i
```

to run the app type:
```
node .\server.js
```

### Prerequisites

Install [node](https://nodejs.org/it/download/)
